### How to deploy on vercel
1. Click the Deploy Button.
2. Set environment variables follow [doc/env-vars.md](doc/env-vars.md).
3. Pay attention to the NEXT_PUBLIC_CHAT_FILES_UPLOAD_PATH value must be /tmp.
